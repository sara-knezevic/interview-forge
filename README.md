### Interview Forge
A comprehensive repository for preparing for technical interviews in software engineering and data science/engineering. This repo contains curated resources, including books, coding problems, and solutions in Python, SQL, and Jupyter Notebooks.

### Repository Structure

`/books`: A collection of essential books and PDFs for technical interview preparation and skill-building.

`/algorithms`: Solutions to algorithmic problems (Leetcode/Neetcode/HackerRank) in Python.

`/python`: Python-specific coding problems and practice scripts.

`/sql`: SQL query problems and solutions for database-related interview questions.

`/notebooks`: Jupyter Notebooks for data science, machine learning, and algorithmic explorations.

### Books
The `/books` folder contains the following resources to support interview preparation:

#### Data Science & Machine Learning:
- _Ace the Data Science Interview_: Practical advice and problems for data science interviews.
- _Andrew Ng's ML Course Notes_: Notes from Andrew Ng's renowned Machine Learning course.
- _Elements of Statistical Learning_: In-depth statistical learning concepts for advanced data science prep.

#### Algorithms & Data Structures:
- _A Common-Sense Guide to Data Structures and Algorithms_: Beginner-friendly guide to core concepts.
- _CLRS Introduction to Algorithms_: Comprehensive reference for algorithm design and analysis.
- _The Algorithm Design Manual_: Practical algorithms with real-world applications.
- _Cracking the Coding Interview_: Classic guide with 189 programming questions and solutions.

#### Python Programming:
- _Effective Python_: Best practices for writing efficient Python code.
- _Fluent Python_: Deep dive into Python's advanced features and idioms.
- _Python Programming Interviews Exposed_: Python-specific interview strategies and problems.

#### System Design & Architecture:
- _Designing Data-Intensive Applications_: Principles for building scalable, reliable systems.
- _Streaming Systems_: Insights into real-time data processing systems.
- _System Design Interview_: Guide to tackling system design interview questions.

#### Miscellaneous:
- _Tech Interview Notebook_: Custom notes and resources for tech interview prep, based on the _Tech Interview Handbook_ & _Cracking the Coding Interview_.
